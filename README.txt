Identity Dossier Builder

Uso rápido:
1. Abrí index.html en Chrome/Edge o subilo a GitHub Pages.
2. Cargá 20-60 fotos.
3. Tocá "auto asignar mejores referencias".
4. Revisá roles manualmente.
5. Descargá identity_dossier.zip y pasalo a ChatGPT junto al prompt.

Nota: usa CDN de MediaPipe Tasks Vision y JSZip, por lo que necesita internet la primera vez para cargar modelos/librerías.
